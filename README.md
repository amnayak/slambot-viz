# 2D Autonomous Mapping: Visualization

Web application made for 18-500: ECE Design Experience at Carnegie Mellon University.

## Quick Setup guide
1) To run the client:

    `cd client`
    
    `npm install`
    
    `npm run start`
    
    The client runs on `localhost:3000`
    

2) To run the server:

    `cd server`
    
    `npm install`
    
    `node app.js`
    
    The server runs on `localhost:3001`
